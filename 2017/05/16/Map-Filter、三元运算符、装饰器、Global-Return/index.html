Map&Filter、三元运算符、装饰器、Global&Return

发表于 2017-05-16 | 分类于 Python_Notes

上周继续了解Python知识里面的Map和Filter、三元运算符、装饰器、Global和Return。同样也是阅读了Python tips，下面将一步步分享我的笔记总结。

一、Map和Filter

Map函数作用是将一个函数映射到一个输入列表的所有元素上。

Filter函数能创建一个列表，其中返回列表里面的每个元素都能对函数返回True值。Filter函数类似于一个For循环，但是它是一个内置函数，并且更快。
