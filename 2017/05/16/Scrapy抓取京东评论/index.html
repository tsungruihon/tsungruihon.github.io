Scrapy抓取京东评论

发表于 2017-05-16 | 分类于 Scrapy爬虫

最近用Scrapy写了一个抓取京东评论的爬虫。其实京东评论爬虫的难度不大，因为通过抓包分析，发现京东专门有一个url专门存放评论信息，而且json格式。这简直是爬虫的最佳数据格式。

http://club.jd.com/comment/productPageComments.action?callback=&productId=3014239&score=0&sortType=5&page=0&pageSize=10&isShadowSku=0

但有个很奇怪的地方就是，京东评论每一页只有10条评论，但是没关系，用一个循环就搞定了。代码放在github上，有兴趣可以了解一下。https://github.com/tsungruihon/jd_comment_scrapy
