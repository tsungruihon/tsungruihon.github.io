Enumerate、对象自省、推导式、异常、lambda和一行式

发表于 2017-05-27 | 分类于 Python_Notes

这周继续深入了解Python知识，下面就Enumerate、对象自省、推导式、异常、lambda表达式和一行式进行笔记总结。

一、Enumerate

Enumerate作为Python内置函数，很特别的地方在于，它允许在遍历数据的时候自动计数。

>>> for counter, value in enumerate([1,2,3,4,5]):


...    print(counter,value)


...    


(0, 1)


(1, 2)


(2, 3)


(3, 4)


(4, 5)

Enumerate可以接受一些可选参数。譬如允许我们定制从哪个数字开始枚举。

>>> my_list = ['apple','banana','grapes','pear']


>>> for c, value in enumerate(my_list, 1):


...    print(c, value)


...    


(1, 'apple')


(2, 'banana')


(3, 'grapes')


(4, 'pear')

同样，可以用来创建包含索引的元组列表。

>>> my_list = ['apple','banana','grapes','pear']


>>> counter_list = list(enumerate(my_list,1))


>>> print(counter_list)


[(1, 'apple'), (2, 'banana'), (3, 'grapes'), (4, 'pear')]

二、对象自省

自省(Introspection)，在计算机编程领域里，是指在运行时来判断一个对象类型的能力，也就说它是什么、它跟神什么相干以及它能做什么。Python中所有一切都是对象，可以仔细勘察那些对象。Python中包含许多内置函数和模块来帮助我们。

dir

dir是用于自省的最重要函数之一。它返回一个列表，列出了一个对象所拥有的属性和方法。

>>> my_list = [1,2,3]


>>> dir(my_list)

如果在运行dir()不传入参数，那么它会返回当前作用域的所有名字。

type & id

type()函数返回一个对象的类型。

>>> print(type(''))


<type 'str'>


>>> print(type([]))


<type 'list'>


>>> print(type({}))


<type 'dict'>


>>> print(type(dict))


<type 'type'>


>>> print(type(3))


<type 'int'>

id()函数返回任意不同种类对象的唯一ID。

>>> name = 'Peter'


>>> print(id(name))


4429242848

issubclass

issubclass(sub, sup)

如果给出的子类sub确定是父类sup的一个子类，则返回True。

hasattr

hasattr(object, name)

检查object是否有一个名为name的值的属性，返回一个布尔值。第二个参数，即属性名字符串的方式提供。该函数一般用于访问某属性前先做一个检查。

getattr

getattr(object, name[, default])

返回obect中名为name值的属性的值。例如如果属性名为‘bar’，则返回obj.bar，该函数会在试图获取一个不存在的属性时，引发AttributeError异常，除非可选的默认参数default。

setattr

setattr(object, name, value)

将给object对象中名为name的值的属性赋值为val。例如如果name为’bar’，则相当于obj.bar = val。如果object中有name的属性则重新设置其值，如果没有则添加一个名为name的属性。

三、推导式

推导式（又称解析式）是Python的一种独有特性，可以从一个数据序列构建另一个新的数据序列的结构体。共有三种推导。

  • 列表(list)推导式
  • 字典(dict)推导式
  • 集合(set)推导式

列表推导式(list comprehensions)

列表推导式提供了一种简明扼要的方法来创建列表。

它的结构是在一个中括号里包含一个表达式，然后是一个for语句，然后是0个或多个for或者if语句。返回结果将是一个新的列表。

>>> multiples = [i for i in range(30) if i % 3 is 0]


>>> print(multiples)


[0, 3, 6, 9, 12, 15, 18, 21, 24, 27]

字典推导式(dict comprehensions)

举一个例子，即利用字典推导式快速对换一个字典的键和值。

>>> {v: k for k, v in mcase.items()}


{3: 'Z', 7: 'A', 10: 'a', 34: 'b'}

集合推导式(set comprehensions)

它们跟列表推导也是类似的。

>>> squared = {x **2 for x in [1,2,3]}


>>> print(squared)


set([1, 4, 9])

四、异常

最熟悉的是try/except从句，我们把可能触发异常发生的代码会放到try语句块里，而处理异常的代码会在except语句块里实现。

处理多个异常

有三种方法来处理多个异常。

第一种方法需要把所有可能发生的异常放到一个元组里。

try:


    file = open('test.txt', 'rb')


except (IOError, EOFError) as e:


    print("An errror occurred. {}".format(e.args[-1]))

另外一种方式是对每个单独的except语句块中处理。

try:


    file = open('test.txt', 'rb')


except EOFError as e:


    print("An EOF error occurred.")


    raise e


except IOError as e:


    print("An error occurred.")


    raise e

最后一种方法是会捕获所有异常：

try:


    file = open('test.txt', 'rb')


except Exception:


    raise

finally

一般而言是把主程序代码包裹进try从句。然后把一些代码包裹进一个except从句，它会在try从句中的代码触发异常时执行。

然后还有finally从句，包裹到finally从句中的代码不管异常是否触发都将会被执行。这可以被用来在脚本执行之后做清理工作。

try:


    file = open('test.txt','rb')


except IOError as e:


    print('An IOError occurred. {}'.format(e.args[-1]))


finally:


    print("This would be printed whether or not an exception occured!")

五、lambda

lambda表达式是一行函数。它们在其他语言中被称为匿名函数。

原型

lambda argument: manipulate(argument)

例子

>>> add = lambda x,y: x+y


>>> print(add(3,5))


8

列表排序

>>> a = [(1,2), (4,1), (9,10), (13, -3)]


>>> a.sort(key=lambda x:x[1])


>>> print(a)


[(13, -3), (4, 1), (1, 2), (9, 10)]

六、一行式

下面是python一些一行式命令。

打印

可以用pprint漂亮的打印出列表和字典，这种方法在字典上更为有效。

>>> from pprint import pprint


>>> my_dict = {'name': 'Yasoob', 'age': 'undefined', 'personality': 'awesome'}


>>> pprint(my_dict)


{'age': 'undefined', 'name': 'Yasoob', 'personality': 'awesome'}

下面是想快速漂亮地从文件打印出json数据，可以这样做

cat file.json | python -m json.tool

CSV to json

python -c "import csv,json;print json.dumps(list(csv.reader(open('csv_file.csv'))))"

列表辗平

可以通过使用itertools包中的itertools.chain.from_iterable轻松快速的辗平一个列表。

>>> import itertools


>>> a_list = [[1,2],[3,4],[5,6]]


>>> print(list(itertools.chain.from_iterable(a_list)))


[1, 2, 3, 4, 5, 6]


>>> print(list(itertools.chain(*a_list)))


[1, 2, 3, 4, 5, 6]

一行式的构造器

class A(object):


    def __init__(self, a, b, c, d, e, f):


        self.__dict__.update({k: v for k, v in locals().items() if k != 'self'})
