Support Vector Machine

发表于 2017-12-26 | 分类于 Machine Learning

今天来让我来解决Support Vector Machine(SVM)。这个算法一直困扰着我，虽然我没有在实际中使用过它，但是我早已听闻它的强大之处。但因为里面充满着各种数学推导，不花一些时间去研究实在是无法消化。但是，解决恐惧的最好方法就是直面它，所以过去三天除了工作外我一直都在看各种相关书籍，以及Youtube上各种SVM的视频，可谓是受尽各种煎熬，当中可能会因为看不懂里面某个公式而坐立不安，然后又找不到人可以沟通请教。所以基本上这个SVM算法整天都在我的脑袋里面挥之不去，然后又没办法完全理清楚每一个细节，这样导致我没办法用代码复写一遍。整个人非常痛苦。但现在，我终于可以来写这个SVM分享文章了。

Support Vector Machine(SVM)，中文是支持向量机，是一种二分类模型。它的基本模型是定义在特征空间上的间隔最大的线性分类器。为什么要强调间隔最大？ 是因为能够将训练样本正确分开的划分超平面可能有很多，但是如何衡量哪一个是最好？间隔最大这一特性使SVM有别于感知机，因为间隔最大可以使对未见示例的泛化能力最强，而且保证对训练样本局部扰动的“容忍”性最好。

支持向量机主要分为以下三类线性可分支持向量机与硬间隔最大化、线性支持向量机与软间隔最大化以及非线形支持向量机与核函数。我一开始看这三个类别的时候，我心里简直就是想爆粗。不知道读者你们有没有类似的感受，如果你们没有，那么下文你们就不需要看了。对于跟我一样感受的读者，我们一起来看看到底这三类支持向量机到底在搞些什么毛线。

我们先来看看我们目前有什么。

在样本空间中，划分超平面可以通过以下线性方程来描述： \[ \mathbf{w}^{T}\mathbf{x} + b = 0 \] 其中， w=(w1; w2; ...; wd)为法向量，决定了超平面的方向；b为位移项，决定了超平面与原点之间的距离。然后样本空间中任意点x到超平面(w, b)的距离可写为： \[ r = \frac {|\mathbf{w}^{T}\mathbf{x} + b|}{||\mathbf{w}||} \] 假设超平面能够正确将训练样本分类，则有： \[ \left\{ \begin{array}{c} \mathbf{w}^{T}\mathbf{x} + b \ge +1, \ \ \ y_i=+1. \\ \mathbf{w}^{T}\mathbf{x} + b \le -1 \ \ \ y_i=-1.\\ \end{array} \right. \] 等价于 \[ y_i\cdot(\mathbf{w}^{T}\mathbf{x} + b) \ge +1 \] 距离超平面最近的几个训练样本能够使等号成立，它们被称为“支持向量(Support Vector)”，两个异类支持向量距离超平面的距离之和为： \[ \gamma= \frac{2}{||\mathbf{w}||} \]

一、线性可分支持向量机与硬间隔最大化

对于线性可分支持向量机，有一个很好的预期就是训练数据的分布没有任何冲突或者混杂，也就是说，这个训练数据集是线性可分。我们需要找出能够将两类数据正确划分并且间隔最大的直线。

构造并求解约束最优化问题： \[ \begin{aligned} & \mathop{min}_{w, b} \ \ \ \frac{1}{2}||w||^2 \\ \\ & s.t. \ \ \ y_i\cdot(\mathbf{w}^{T}\mathbf{x} + b) - 1\ge 0 \end{aligned} \] 然后构建拉格朗日函数: \[ L(w, b, \alpha) = \frac{1}{2}||w||^2 - \sum_{i=1} ^{N} \alpha_i\cdot((\mathbf{w}^{T}\mathbf{x} + b) - 1) \] 因此，问题变成极大极小问题： \[ \mathop{max}_\alpha\mathop{min}_{w, b} L(w, b, \alpha) \] （1）求min_{w, b}L(w, b, a) \[ \begin{aligned} \frac {\partial L}{\partial w} & = w - \sum_{i}^{N}\alpha_iy_ix_i = 0 \\ \\ \frac {\partial L}{\partial b} & = \sum_{i}^{N}\alpha_iy_i = 0 \end{aligned} \] 得 \[ \begin{aligned} & w = \sum_{i}^{N}\alpha_iy_ix_i \\ \\ & \sum_{i}^{N}\alpha_iy_i = 0 \end{aligned} \] 所以将这两个式子代入L(w, b, a)得： \[ \mathop{min}_{w, b} L(w, b, \alpha) = \sum_{i=1}^{N}\alpha_i - \frac {1}{2}\sum_{i=1}^{N}\sum_{j=1}^{N}\alpha_i\alpha_jy_iy_j(x_i \cdot x_j) \] （2）求min_{w, b} L(w, b, a)对a的极大： \[ \mathop{max}_{\alpha} L(w, b, \alpha) = \sum_{i=1}^{N}\alpha_i - \frac {1}{2}\sum_{i=1}^{N}\sum_{j=1}^{N}\alpha_i\alpha_jy_iy_j(x_i \cdot x_j) \] 这个问题的对偶问题是： \[ \begin{aligned} & \mathop{min}_{\alpha} \ \ \ L(w, b, \alpha) =\frac {1}{2}\sum_{i=1}^{N}\sum_{j=1}^{N}\alpha_i\alpha_jy_iy_j(x_i \cdot x_j) - \sum_{i=1}^{N}\alpha_i \\ & s.t \ \ \ \sum_{i=1}^{N}\alpha_iy_i = 0 \\ \\ & \ \ \ \ \ \ \ \ \ \alpha_i \ge 0 \\ \\ & \ \ \ \ \ \ \ \ \ y_i\cdot(\mathbf{w}^{T}\mathbf{x} + b) - 1\ge 0 \\ \\ & \ \ \ \ \ \ \ \ \ \alpha_i(y_i\cdot(\mathbf{w}^{T}\mathbf{x} + b) - 1) = 0 \end{aligned} \] 求得最优解a*=(a1*, a2*, ..., aN*)^T

（3）计算 \[ w^{*} = \sum_{i=1}^{N}\alpha_iy_ix_i \] 并选择a*的一个正分量aj*>0，计算得 \[ b^{*} = y_j - \sum_{i=1}^{N}\alpha_{j}^{*}y_i(x_i \cdot x_j) \] （4）求得分离超平面 \[ w^{*} \cdot x + b^{*} = 0 \] （5）分类决策函数 \[ f(x) = sign(w^{*} \cdot x + b^{*}) \] 解决了。

在线性可分支持向量机中，w*和b*只依赖于训练数据中对应aj*>0的样本点(xi, yi)，而其他样本点对w*和b*没有影响。因此我们把训练数据中对应aj*>0的实例点称为支持向量。

二、线性支持向量机与软间隔最大化

但是训练数据不一定总是线性可分，有时候某些样本点不一定满足间隔大于等于1得约束条件。因此，我们需要修改硬间隔最大化为软间隔最大化。也就是说，我们希望在最大化间隔的同时，不满足约束条件的样本应尽可能少。

于是优化目标可写为 \[ \mathop{min}_{w, b} \frac{1}{2}||w||^2 + C\sum_{i=1}^{m}\ell_{0/1}(y_i(\mathbf{w}^{T}\mathbf{x} + b )-1) \] 其中C>0使一个常数，𝓁0/1是“0/1损失函数” \[ \ell_{0/1}{(z)}= \begin{cases} 1, & \text{if $z$ <0;} \\[2ex] 0, & \text{otherwise.} \end{cases} \] 若C为无穷大时，这样会迫使所有样本都要满足原始约束。当C取有限值时，则允许一些样本不满足约束。

但𝓁0/1非凸、非连续，数学性质不太好，使得不易直接求解。于是常用其他替代损失函数来代替𝓁0/1。它们通常是凸的连续函数且是𝓁0/1的上界。 \[ \begin{aligned} & hinge \ \ loss: \ell_{hinge}(z) = max(0, 1-z) \\ \\ & exponential \ \ loss: \ell_{exp}(z) = exp(-z) \\ \\ & logistic \ \ loss: \ell_{log}(z) = log(1 + exp(-z)) \end{aligned} \]

引入松弛变量(slack variables)， 对每个样本点(xi, yi)引进一个松弛变量𝜉i>=0，使函数间隔加上松弛变量大于等于1。这时候，约束条件变为： \[ y_i(w \cdot x_i + b ) + \xi_i \ge 1 \]

同时，对每个松弛变量𝜉i， 支付一个代价𝜉i。这时候目标函数变成： \[ \frac {1}{2} ||w||^2 + C \sum_{i=1}^{N}\xi_{i} \] 这里，C>0称为惩罚参数，一般由应用问题决定。C值大时对误分类的惩罚增大，C值小时对误分类的惩罚减少。目标函数有两层含义，第一层是希望间隔尽可能大，同时使误分类点的个数尽量小，C是调和二者的系数。

构造并求解约束最优化问题： \[ \begin{aligned} & \mathop{min}_{w, b, 𝜉}\ \ \frac {1}{2} ||w||^2 + C \sum_{i=1}^{N}\xi_{i} \\ \\ & s.t. \ \ y_i(w \cdot x_i + b ) + \xi_i \ge 1 \\ \\ & \xi_i \ge 0, \ \ i=1, 2, ..., m. \end{aligned} \] 然后构建拉格朗日函数： \[ \begin{aligned} L(\mathbf{w}, b, \mathbf{\alpha}, \xi, \mu) = \mathop{min}_{w, b, 𝜉}\ \ & \frac {1}{2} ||w||^2 + C \sum_{i=1}^{N}\xi_{i} \\ & +\sum_{i=1}^{m}\alpha_i(1- \xi_i - (\mathbf{w}^{T}\mathbf{x} + b )-1)) - \sum_{i=1}^{m}\mu_i\xi_i \end{aligned} \] 其中a>=0，ui>=0是拉格朗日乘子。

（1）令L(w, b, a, 𝜉, u)对w, b, 𝜉的偏导为零可得： \[ \begin{aligned} \mathbf{w} & = \sum_{i=1}^{N}\alpha_iy_ix_i, \\ \\ 0 &= \sum_{i=0}^{m} \alpha_iy_i \\ C & = \alpha_i + \mu_i \end{aligned} \] （2）求min{w, b, 𝜉} L(w, b, a, 𝜉, u)对a的极大： \[ \begin{aligned} & \mathop{min}_{\alpha} \ \ \ \frac {1}{2}\sum_{i=1}^{m}\sum_{j=1}^{m}\alpha_i\alpha_jy_iy_j(x_i \cdot x_j) - \sum_{i=1}^{N}\alpha_i \\ & s.t \ \ \ \sum_{i=1}^{m}\alpha_iy_i = 0\\ \\ & \ \ \ \ \ \ \ \ \ 0 \le \alpha_i \le C , \ \ \ \mu_i \ge 0, \\ \\ & \ \ \ \ \ \ \ \ \ y_i\cdot(\mathbf{w}^{T}\mathbf{x} + b) - 1 + \xi_i \ge 0 \\ \\ & \ \ \ \ \ \ \ \ \ \alpha_i(y_i\cdot(\mathbf{w}^{T}\mathbf{x} + b) - 1+ \xi_i ) = 0 \\ \\ & \ \ \ \ \ \ \ \ \ \xi_i \ge 0, \ \ \ \mu_i\xi_i = 0 \end{aligned} \] （3）计算 \[ w^{*} = \sum_{i=1}^{N}\alpha_iy_ix_i \] 并选择a*的一个正分量0<=aj*<=C，计算得 \[ b^{*} = y_j - \sum_{i=1}^{N}\alpha_{j}^{*}y_i(x_i \cdot x_j) \] （4）求得分离超平面 \[ w^{*} \cdot x + b^{*} = 0 \] （5）分类决策函数 \[ f(x) = sign(w^{*} \cdot x + b^{*}) \] 解决了。

三、非线形支持向量机与核函数

对于线性分类问题，线性分类支持向量机是一种非常有效的方法，但是，有时候分类问题是非线性的。

这时候，想用一条直线去把这两类数据进行正确分类是不现实的。因此我们可以使用非线性支持向量机。

非线性支持向量机首先使用一个变换将原空间的数据映射到新空间；然后在新空间里用线性分类学习方法从训练数据中学习分类模型。实现这个过程最主要的方法就是核函数(Kernel Function)。

在原属性空间中计算的相似度函数K称为核函数(Kernel Function)，在满足Mercer原理，也就是说必须存在一个相应的变换，使得计算一对向量的核函数等价于在变换后的空间中计算这对向量的点积，能够使计算点积的开销更小，而且能够避免维灾难。也就是说，有 \[ \mathbf{K}(\mathbf{u, \mathbf{v}}) = \mathbf{\Phi}{(\mathbf{u})} \cdot \mathbf{\Phi}{(\mathbf{v})} = (\mathbf{u} \cdot \mathbf{v} + 1) ^ 2 \] 下面是一些常用的核函数

  • 多项式核函数（polynomial kernel function)

\[ K(x, z) = (x \cdot z + 1)^p \]

对应的支持向量机是一个p次多项式分类器。

  • 高斯核函数(Gaussian kernel function) \[ K(x, z) = exp\left(-\frac{||x-z||^2}{2\sigma^2}\right) \] 然后下面我们就可以尝试求解最优解了。

（1）选取适当的核函数K(x, z)和适当的参数C，构造并求解最优化问题 $$ \begin{aligned} & {}    {i=1}^{m}_{j=1}^{m}_ijy_iy_j(K(x_i, x_j)) - {i=1}{N}i \ & s.t     {i=1}{m}_iy_i = 0\ \ &           0 _i C ,     i =1, 2, …, m \

\end{aligned} $$ 求得最优解ɑ*=(ɑ1*, a2*, ..., a*N)^T

（2）选择ɑ*的一个正分量0<ɑj*<C，计算 \[ b^* = y_j - \sum_{i=1}^{N}\alpha_{i}^{*}y_iK(x_i \cdot x_j) \] （3）求得分离超平面 \[ \sum_{i=1}^{N}\alpha_{i}^{*}y_iK(x \cdot x_i) + b^{*} = 0 \] （4）分类决策函数 \[ f(x) = sign \left (\sum_{i=1}^{N}\alpha_{i}^{*}y_iK(x \cdot x_i) + b^{*}\right) \] 解决了。
