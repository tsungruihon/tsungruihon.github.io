首先，监督学习是指在有标记的样本(labeld samples)上建立机器学习的模型。有了这些带标记的数据，算法就可以学会如何根据输入的参数计算。无监督血虚面对的是没有标记的数据。假设需要把一些数据分成不同的组别，但是对分组的条件毫不知情，于是，无监督学习算法就会以最合理的方式将数据集分成确定数量的组别。

一、数据预处理技术

为了让机器学习算法理解原始数据，需要对数据进行预处理。

1.均值移除

通常会把每个特征的平均值移除，以保证特征均值为0（即标准化处理）。这样做可以消除特征彼此的偏差（bias）。

import numpy as np

3.归一化（Normalization）

数据归一化用于对特征向量的值进行调整，保证每个特征向量的值都缩放到相同的数值范围。确保数据点没有因为特征的基本性质而产生较大差异，确保数据处于同一数量级，提高不同特征数据的可比性。

常用的归一化形式就是将特征向量调整为L1范数，使特征向量的数值之和为1。L0范数是指向量中非0的元素个数。如果用L0范数来规则化一个参数矩阵W，会希望W大部分元素都是0。L1范数和L2范数都可以实现矩阵稀疏，L1因具有比L0更好的优化求解特征而被广泛应用。而参数稀疏能够更好的进行特征选择以及增强解释性。L2范数在回归里面也叫“岭回归(Ridge Regression)”，也被称为权值衰减weight decay。岭回归能够改善模型过拟合问题。在L2范数中，限制了参数的变化波动范围，降低了标准差。总的而言，L2范数可以防止过拟合，提升模型的泛化能力。

data_normalized = preprocessing.normalize(data, norm='l1')


print "\nL1 normalized data =", data_normalized

四、计算回归准确性

现在建立了回归器，接下来就是评价回归器的拟合效果。

  • 平均绝对误差（Mean Absolute Error）：给定数据集的所有数据点的绝对误差平均值。

  • 均方误差（Mean Squared Error）：给定数据集的所有数据点的误差的平方的平均值。

  • 中位数绝对误差（Median Absolute Error）：给定所有数据集的所有数据点的误差的中位数。这个指标的主要优点是可以消除异常值的干扰。

  • 解释方差分（Explained Variance Score）：用于衡量模型对数据集波动的解释能力，如果得分1.0分，表明模型是完美的。

  • R方得分（R2 Score）：指确定性相关系数，用于衡量模型对未知样本预测的效果。最好的得分是1.0。

    一般都是保证均方误差最低，解释方差分最高。

import sklearn.metrics as sm




print "Mean absolute error =", round(sm.mean_absolute_error(y_test, y_test_pred), 2)


print "Mean squared error =", round(sm.mean_squared_error(y_test, y_test_pred), 2)


print "Median absolute error =", round(sm.median_absolute_error(y_test, y_test_pred), 2)


print "Explained variance score =", round(sm.explained_variance_score(y_test, y_test_pred), 2)


print "R2 score =", round(sm.r2_score(y_test, y_test_pred), 2)

五、创建岭回归器

七、估算房屋价格

下面将使用AdaBoost算法的决策树回归器(decision tree regressor)来解决房屋估价这一问题。

决策树是一个树状模型，每个节点都做出一个决策，从而影响结果。叶子节点表示输出数值，分支表示根据输入特征做出的中间决策。Adaboost算法是指自适应增强(Adaptive boosting)算法，这是一种利用其他系统增强模型准确性的技术。这种技术将不同版本的算法结果进行组合，用加权汇总的方式获得最终结果，被称为弱学习(weak learners)。Adaboost算法在每个阶段获取的信息都会反馈到模型中，这样学习器就可以在后一阶段重点训练难以分类的样本。这种学习方式可以增强系统的准确性。

首先使用Adaboost算法对数据集合进行回归拟合，再计算误差，然后根据误差评估结果，用同样的数据集重新拟合。可以把这些看作是回归器的调优过程，直到达到预期的准确性。

def plot_feature_importance(feature_importances, title, feature_names):


    # Normalize the importance values


    feature_importances = 100.0 * (feature_importances / max(feature_importances))


    


    # 将得分从高到低排序


    index_sorted = np.flipud(np.argsort(feature_importances))


    


    # Arrange the X ticks


    pos = np.arange(index_sorted.shape[0]) + 0.5


    


    # Plot the bar graph


    plt.figure()


    plt.bar(pos, feature_importances[index_sorted], align='center')


    plt.xticks(pos, feature_names[index_sorted])


    plt.ylabel('Relative Importances')


    plt.title(title)


    plt.show()
