ML读书笔记一

发表于 2017-09-05 | 分类于 Machine Learning

有好长一段时间没有更新博客，前段时间一直都在看机器学习的相关书籍，但一直都没有看太懂。第一本看的是跟贝叶斯概率相关的，印象最深刻的是里面讲了一个pymc的库，很多细节参数书本上并没有做很详尽的解释，大多数时候就直接讲如何应用，虽然见识了这个库的强大，但有很多地方和思路都无从适应。第二本书讲的是Python机器学习里面一些基本概念和应用，从监督学习无监督学习到线性回归里面的最小二乘法和岭回归，到了后面的集成学习方法。这本书我基本上就是代码都敲过一遍，总体上摸了一些思路。书本把一些模型的细节讲得很细，似懂非懂。但无论如何，被这两本书虐完之后，我总算对机器学习的认识相比以前没有那么模糊，

现在我的学习计划就是，看一章周志华老师的《机器学习》，把概念数学公式全部弄清楚，不着急看下一章。然后在《Python机器学习经典实例》把对应的案例和应用都学一遍，做到学以致用。然后在Youtube上看李宏毅教授对应的ML课程，最后在《Python数据分析实战》看一章，巩固Python在数据分析里面库的应用。然后再《机器学习》—>《Python机器学习经典案例》—>Youtube中李宏毅教授对应的ML课程—>《Python数据分析实战》循环。目前在进行第一个循环，刚刚刷了《Python机器学习经典案例》的第一章线性回归，目前两个过程还是受益匪浅。所以就在这里记录我的学习笔记。

首先，监督学习是指在有标记的样本(labeld samples)上建立机器学习的模型。有了这些带标记的数据，算法就可以学会如何根据输入的参数计算。无监督血虚面对的是没有标记的数据。假设需要把一些数据分成不同的组别，但是对分组的条件毫不知情，于是，无监督学习算法就会以最合理的方式将数据集分成确定数量的组别。

一、数据预处理技术

为了让机器学习算法理解原始数据，需要对数据进行预处理。

1.均值移除

通常会把每个特征的平均值移除，以保证特征均值为0（即标准化处理）。这样做可以消除特征彼此的偏差（bias）。

import numpy as np
from sklearn import preprocessing

data = np.array([
  [3, -1.5, 2, -5.4],
  [0, 4, -0.3, 2.1],
  [1, 3.3, -1.9, -4.3]
])
data_standardized = preprocessing.scale(data)
print "\nMean =", data_standardized.mean(axis=0)
print "\nStd deviation =", data_standardized.std(axis=0)

3.归一化（Normalization）

数据归一化用于对特征向量的值进行调整，保证每个特征向量的值都缩放到相同的数值范围。确保数据点没有因为特征的基本性质而产生较大差异，确保数据处于同一数量级，提高不同特征数据的可比性。

常用的归一化形式就是将特征向量调整为L1范数，使特征向量的数值之和为1。L0范数是指向量中非0的元素个数。如果用L0范数来规则化一个参数矩阵W，会希望W大部分元素都是0。L1范数和L2范数都可以实现矩阵稀疏，L1因具有比L0更好的优化求解特征而被广泛应用。而参数稀疏能够更好的进行特征选择以及增强解释性。L2范数在回归里面也叫“岭回归(Ridge Regression)”，也被称为权值衰减weight decay。岭回归能够改善模型过拟合问题。在L2范数中，限制了参数的变化波动范围，降低了标准差。总的而言，L2范数可以防止过拟合，提升模型的泛化能力。

data_normalized = preprocessing.normalize(data, norm='l1')
print "\nL1 normalized data =", data_normalized

5.独热编码

独热编码可以看作是一种收紧（tighten）特征向量的工具。它把特征向量的每个特征与特征的非重复总数相对应，通过one of k的形式对每个值进行编码。特征向量的每个特征值都按照这种方式编码，能够更有效地表示空间。

encoder = preprocessing.OneHotEncoder()
encoder.fit([[0, 2, 1, 12], [1, 3, 5, 3], [2, 3, 2, 12], [1, 2, 4, 3]])

四、计算回归准确性

现在建立了回归器，接下来就是评价回归器的拟合效果。

  • 平均绝对误差（Mean Absolute Error）：给定数据集的所有数据点的绝对误差平均值。

  • 均方误差（Mean Squared Error）：给定数据集的所有数据点的误差的平方的平均值。

  • 中位数绝对误差（Median Absolute Error）：给定所有数据集的所有数据点的误差的中位数。这个指标的主要优点是可以消除异常值的干扰。

  • 解释方差分（Explained Variance Score）：用于衡量模型对数据集波动的解释能力，如果得分1.0分，表明模型是完美的。

  • R方得分（R2 Score）：指确定性相关系数，用于衡量模型对未知样本预测的效果。最好的得分是1.0。

    一般都是保证均方误差最低，解释方差分最高。

import sklearn.metrics as sm

print "Mean absolute error =", round(sm.mean_absolute_error(y_test, y_test_pred), 2)
print "Mean squared error =", round(sm.mean_squared_error(y_test, y_test_pred), 2)
print "Median absolute error =", round(sm.median_absolute_error(y_test, y_test_pred), 2)
print "Explained variance score =", round(sm.explained_variance_score(y_test, y_test_pred), 2)
print "R2 score =", round(sm.r2_score(y_test, y_test_pred), 2)

五、创建岭回归器

线性回归的主要问题是对异常值敏感，由于异常值误差的绝对值很大，因此会引起问题，从而破坏整个模型。为了避免这个问题，利用岭回归引入正则化项的系数作为阈值来消除异常值的影响。

import sys

import numpy as np

filename = sys.argv[1]
X = []
y = []
with open(filename, 'r') as f:
    for line in f.readlines():
        data = [float(i) for i in line.split(',')]
        xt, yt = data[:-1], data[-1]
        X.append(xt)
        y.append(yt)
        
num_training = int(0.8 * len(X))
num_test = len(X) - num_training

X_train = np.array(X[:num_training])
y_train = np.array(y[:num_training])

from sklearn import linear_model

linear_regressor = linear_model.LinearRegression()
# alpha参数控制回归器的复杂程度，当alpha趋于0时，岭回归就是用普通最小二乘法的线形回归器
# 如果希望模型对异常值不那么敏感，就需要设置一个较大的alpha值
ridge_regressor = linear_model.Ridge(alpha=0.01, fit_intercept=True, max_iter=10000)

linear_regressor.fit(X_train, y_train)
ridge_regressor.fit(X_train, y_train)

y_test_pred = linear_regressor.predict(X_test)

六、创建多项式回归器

线性回归模型有一个主要的局限性，就是只能把输入数据拟合成直线，而多项式回归模型通过拟合多项式方程式来克服这类问题，从而提高模型的准确性。

from sklearn.preprocessing import PolynomialFeatures

七、估算房屋价格

下面将使用AdaBoost算法的决策树回归器(decision tree regressor)来解决房屋估价这一问题。

决策树是一个树状模型，每个节点都做出一个决策，从而影响结果。叶子节点表示输出数值，分支表示根据输入特征做出的中间决策。Adaboost算法是指自适应增强(Adaptive boosting)算法，这是一种利用其他系统增强模型准确性的技术。这种技术将不同版本的算法结果进行组合，用加权汇总的方式获得最终结果，被称为弱学习(weak learners)。Adaboost算法在每个阶段获取的信息都会反馈到模型中，这样学习器就可以在后一阶段重点训练难以分类的样本。这种学习方式可以增强系统的准确性。

首先使用Adaboost算法对数据集合进行回归拟合，再计算误差，然后根据误差评估结果，用同样的数据集重新拟合。可以把这些看作是回归器的调优过程，直到达到预期的准确性。

import numpy as np
from sklearn.ensemble import RandomForestRegressor, AdaBoostRegressor
from sklearn.tree import DecisionTreeRegressor

from sklearn.metrics import mean_squared_error, explained_variance_score
from sklearn.utils import shuffle
import matplotlib.pyplot as plt

def plot_feature_importance(feature_importances, title, feature_names):
    # Normalize the importance values
    feature_importances = 100.0 * (feature_importances / max(feature_importances))
    
    # 将得分从高到低排序
    index_sorted = np.flipud(np.argsort(feature_importances))
    
    # Arrange the X ticks
    pos = np.arange(index_sorted.shape[0]) + 0.5
    
    # Plot the bar graph
    plt.figure()
    plt.bar(pos, feature_importances[index_sorted], align='center')
    plt.xticks(pos, feature_names[index_sorted])
    plt.ylabel('Relative Importances')
    plt.title(title)
    plt.show()
