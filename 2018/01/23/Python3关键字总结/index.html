Python3关键字总结

发表于 2018-01-23 | 分类于 Python_Notes

这一节主要介绍Python3的关键字。

首先，通过使用import keyword我们可以看到python3的关键字列表。

import keyword
print(keyword.kwlist)

[output]
['False', 'None', 'True', 'and', 'as', 'assert', 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except', 'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda', 'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try', 'while', 'with', 'yield']

下面我们来看看这33个关键字的各种解释。

True, False

True和False主要是比较运算和逻辑运算的结果体现。例如

>>> 1 == 1
True
>>> 5 > 3
True
>>> True or False
True
>>> 10 <= 1
False
>>> 3 > 7
False
>>> True and False
False
>>> True == 1
True
>>> False == 0
True
>>> True + True
2

None

None在Python中是一个特殊的实例(<class 'NoneType'>)，代表着缺失值或者空值。

当函数没有return声明的时候，函数将会返回None

>>> None == 0
False
>>> None == []
False
>>> None == False
False
>>> x = None
>>> y = None
>>> x == y
True
>>> def a_void_function():
...     a = 1
...     b = 2
...     c = a + b
...
>>> x = a_void_function()
>>> print(x)
None

and, or, not

只有操作数都是True，and结果才会输出True。

A B A and B
True True True
True False False
False True False
False False False

Or会返回True只要其中一个操作数是True。

A B A or B
True True True
True False True
False True True
False False False

not操作数用来反转值。

not A A
False True
True False 
>>> True and False
False
>>> True or False
True
>>> not False
True

as

as是用于在导入模块的时候给模块的别称。例如

>>> import math as myAlias
>>> myAlias.cos(myAlias.pi)
-1.0

assert

assert是用来程序调试的。assert语句用来声明某个条件是真的,当assert语句失败的时候，会引发一AssertionError。例如

>>> a = 4
>>> assert a < 5
>>> assert a > 5
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AssertionError

为了更好的理解输出，我们还可以在信息打印中提供更具体的信息。信息会在AssertionError中体现出来。

>>> assert a > 5, "The value of a is too small"
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AssertionError: The value of a is too small

在这里，我们应该可以发现

assert condition, message

等同于

if not condition:
    raise AssertionError(message)

break, continue

break和continue主要用于for和while语句里，用于改变它们正常的行为。

break会结束它所在的最小循环，并且将控制流向循环后面的语句。而continue会使当下的循环结束，但不会影响整个循环。

>>> for i in range(1, 11):
...     if i == 5:
...         break
...     print(i)
...
1
2
3
4
>>> for i in range(1, 11):
...     if i == 5:
...         continue
...     print(i)
...
1
2
3
4
6
7
8
9
10

class

class用于定义类。

类是相关属性和方法的集合，它是用于表示a real world situation。 这种把数据和方法集合在一个类的思想就是面对对象编程的中心思想。

>>> class ExampleClass:
...     def function1(parameters):
...         pass
...     def function2(parameters):
...         pass
...

def

def用来定义函数。它们可以共同完成一些特定的任务，函数能够帮助我们将代码组织成可管理的块，也可以完成一些重复性的任务。

>>> def function1(parameters):
...     pass

del

del用于删除指向对象的指针。

>>> a = b = 5
>>> del a
>>> a
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
NameError: name 'a' is not defined
>>> b
5

同样，del也可以用于将列表或者字典里面的元素删除。

>>> a = ['x', 'y', 'z']
>>> del a[1]
>>> a
['x', 'z']

if, else, elif

用于条件语句。

except, raise, try

except、raise和try是用于处理异常。常见的异常有IOError, ValueError, ZeroDivisionError, ImportError, NameError, TypeError等等。

def reciprocal(num):
    try:
        r = 1/num
    except:
        print('Exception caught')
        return
    return r

print(reciprocal(10))
print(reciprocal(0))

[output]
0.1
Exception caught
None

同时我们还可以显性说明错误。

if num == 0:
    raise ZeroDivisionError('cannot divide')

finally

finally确保代码块的内容会被执行，即使有一个并没有被事先预备的异常。

try:
    Try-block
except exception1:
    Exception1-block
except exception2:
    Exception2-block
else:
    Else-block
finally:
    Finally-block

for

for循环啦，这个就不用多说什么啦。

from, import

主要用于引入模块，这个就不用多说什么啦。

global

global是用来声明在函数里面的变量为全局变量。

如果需要读取全局变量的值，那么其实没有必要声明变量为global。

如果需要在函数里面修改全局变量的值，那我们必须用global来声明它。否则一个相同命名的局部变量就会被创造出来。

>>> globvar = 10
>>> def read1():
...     print(globvar)
...
>>> def write1():
...     global globvar
...     globvar = 5
...
>>> def write2():
...     globvar = 15
...
>>> read1()
10
>>> write1()
>>> read1()
5
>>> write2()
>>> read1()
5

in

in是用来测试一个序列里面是否包含一个对象或者值。如果包含，将会返回True，否则返回False。另外一个作用就是用来结合for遍历序列。

is

is是用来测试对象的标识。==用来判断两个变量是否相等，is用来判断两个变量是否指向同一个对象。

我们知道针对True、False和None只有一个实例，所以它们是独有的。

>>> True is True
True
>>> False is False
True
>>> None is None
True
>>>

一个空的列表或者字典跟空的是相等的，但它们并不是独有的对象因为它们在内存中是单独分配的。这是因为列表和字典是可变对象。

>>> [] == []
True
>>> [] is []
False
>>> {} == {}
True
>>> {} is {}
False

不像列表和字典，字符串和元组都是不可变对象（就是一旦定义后值就不会发生变化）。因此，两个相同的字符串或者元组是相同的，它们都指向相同的内存地址。

lambda

lambda用于创造匿名函数（也就是没有名字的函数）。这是一个不包含return语句的内联函数，它由一个被评估和返回的表达式组成。

>>> a = lambda x: x *2
>>> for i in range(1, 6):
...     print(a(i))
...
2
4
6
8
10

nonlocal

nonlocal的用法和global的用法非常相似。nonlocal用于声明在嵌套函数里面的变量并不属于外部函数。如果我们需要修改在嵌套函数里的非局部变量的值，我们必须用nonlocal来声明。

>>> def outer_function():
...     a = 5
...     def inner_function():
...          nonlocal a
...          a = 10
...          print('Inner function: ', a)
...     inner_function()
...     print('Outer function: ', a)
...
>>> outer_function()
Inner function:  10
Outer function:  10

pass

pass嘛，就是作为一个占位符，当执行它的时候什么都不会发生。

return

return声明用于结束函数并且返回值。如果我们没有显性返回值，那么函数将会自动返回None值。

while

while比较基本啦，可以结合else使用。

with

基本思想是with所求值的对象必须有一个__enter__( )方法，一个__exit__( )方法。

with语句的执行逻辑如下：紧跟with后面的语句被求值后，返回对象的__enter__( )方法被调用，这个方法的返回值将被赋值给as后面的变量。当with后面的代码块全部被执行完之后，将调用前面返回对象的__exit__( )方法。

with open('example.txt', 'w') as my_file:
    my_file.write('Hello world!')

yield

yield在函数里面像return，但yield返回的是一个生成器。生成器是一个一次只能生成一个对象或者值的迭代器。一个很大的列表的会占用非常大的内存，而生成器将会很好的应对这类情况，因为它每次只会生成一个值，也就是实时地生成数据，而不是把所有的值都存放在内存中。

在生成器中，我们可以用next()方法生成值。

>>> g = (2 ** x for x in range(100))
>>> next(g)
1
>>> next(g)
2
>>> next(g)
4
>>>

哇，不知不觉就结束了，结束得毫无征兆。
