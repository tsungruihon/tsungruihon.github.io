FastText优化文档

发表于 2018-04-20

编写时间：2018年4月20日

编写人：曾锐鸿

  • 工作内容

  • 数据

    1. 将C2的inteligence_home的数据增加到C1
    2. 将remind部分数据标签更改为alarm
    3. 将inteligence_home部分数据标签更改为volume
    4. 增加了30条结合alarm + command的数据
    5. 利用异步编程加快hlt分词训练数据收集

  • 模型代码

    1. 用了Facebook官方的fastText(https://github.com/facebookresearch/fastText)包重新fastText训练模型。
    2. 在K折交叉验证的过程中，保留每次验证的confusion_matrix, classification_report, PR值以及记录log文件
    3. 自动输出原数据标签分布以及输出分类器在测试数据上与原数据标签不同的数据（可指定特定标签）

    ​

  • 工具使用

  1. python v_3.6.4
  2. pandas v_0.22.0
  3. fastText v_facebook_official
  4. sklearn v_19.1
